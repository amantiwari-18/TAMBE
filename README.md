# Tambe Global - Precision Engineered Automotive Components

Modern, luxury website for automotive component manufacturing company with complete admin panel.

## 🚀 Quick Start

### Local Development
```bash
# Frontend
cd frontend
yarn install
yarn start

# Backend  
cd backend
pip install -r requirements.txt
uvicorn server:app --reload
```

### Deployment
See [DEPLOYMENT.md](./DEPLOYMENT.md) for complete deployment guide.

## 📦 Tech Stack

- **Frontend**: React, Tailwind CSS, Framer Motion
- **Backend**: FastAPI, Python
- **Database**: MongoDB
- **Hosting**: Vercel (Frontend), Render (Backend)

## 🎨 Features

- ✅ Hero slider management
- ✅ Product catalog with image upload
- ✅ WhatsApp integration
- ✅ Theme customization (colors, fonts)
- ✅ SEO optimized for tambeglobal.com
- ✅ Mobile responsive
- ✅ Admin panel for content management
